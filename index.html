About

SCP Docs は、SCP財団のコンテンツを快適に閲覧・管理するための非公式リーダーアプリです。

Features

・SCP記事の高速閲覧
・読書履歴と進捗管理
・評価・分析機能
・タグ / カテゴリ探索
