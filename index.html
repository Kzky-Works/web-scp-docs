SCP Docs
Secure • Contain • Protect
[ OVERVIEW ]
SCP Docs は、SCP財団関連資料を閲覧・管理するための非公式モバイルアプリケーションです。

当アプリは公開されている資料をもとに構築されています。
[ FUNCTIONAL CAPABILITIES ]
- SCP文書の高速閲覧
- 読書進捗トラッキング
- 評価・記録機能
- タグ / カテゴリ検索
[ SUPPORT CHANNEL ]

your@email.com
[ DATA HANDLING POLICY ]
LOG TYPE: LOCAL + NETWORK + ADS (NO SERVER ACCOUNT)

当アプリは、次の情報を取り扱う場合があります。

端末内に保存される情報
・閲覧履歴、評価、お気に入り／後で読む、読書再開のための進捗、メモ等（本アプリ機能の提供のため）

通信が発生する処理
・記事一覧・カタログ等データの取得
・アプリ内表示のため、第三者が運営するウェブサイト（記事本文）へのアクセス
・App Store を通じたサブスクリプションの購入・更新に伴う処理（Apple が購入情報を管理）
・Google AdMob による広告表示（バナー・ネイティブ・リワード等）。広告配信の過程で、広告識別子等が Google およびそのパートナーに送信されることがあります

診断・改善
・アプリの安定性・改善のため、OS や App Store の仕組みを通じて提供されるクラッシュレポート等が配信元に届く場合があります（本アプリに独自のクラッシュ収集 SDK を組み込んでいない場合もあります）

当アプリは、運営者がユーザーを特定するためのアカウント登録や、氏名・住所等の取得を行うものではありません。一方で、広告・課金・外部サイト閲覧・OS／配信元の仕組みにより、各事業者のポリシーに従い情報が取得・送信される場合があります。詳細は本ページのプライバシーポリシー全文をご参照ください。
