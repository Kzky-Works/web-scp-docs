Overview

Secure · Contain · Read

SCP Docs — это неофициальный фанатский ридер для материалов, связанных с SCP и опубликованных в открытом интернете, включая сайты на базе Wikidot. Приложение собирает точки входа в архивы, состояние чтения, библиотеку и поиск в нативное iOS-пространство, чтобы вы могли возвращаться к отчетам без повторного восстановления пути в браузере.

Design rationale

Сначала нативность, затем особенности платформы

SCP Docs — не универсальная веб-обертка. Текущий релиз является нативным iOS-приложением на технологиях Apple, с поверхностями SwiftUI для навигации, списков архивов, настроек и инструментов чтения. Прокрутка, переходы, тактильная отдача и доступность настроены так, чтобы ощущаться частью платформы.

Официально распространяемый клиент сейчас доступен только для iOS. Дата выхода версии для Android не объявлялась. Если в будущем будет рассматриваться другая платформа, цель останется той же: следовать ее нативным UI-паттернам и жизненному циклу, а не выпускать одну общую оболочку для всех систем.

  • Фанатский проект — SCP Docs независим от SCP Foundation, Wikidot, Inc. и операторов упоминаемых сайтов.
  • Слой ридера — статьи открываются из их публичных URL во встроенном WebKit-ридере; исходные сайты и их лицензии остаются авторитетным источником.

App shell

Рабочее пространство для чтения из четырех вкладок

Основное приложение организовано вокруг Главной, Библиотеки, Поиска и Настроек; состояние чтения переносится между архивными разделами.

Главная
Контекст филиала, точки входа в архивы, категории, случайные открытия и инструменты прямого перехода.
Библиотека
Избранное, история, список «прочитать позже», личные оценки и структурированные маршруты к Tales, GoI, Canon, Joke и связанным спискам.
Поиск
Поиск по синхронизированным метаданным каталогов, включая названия, теги, классы объектов, заметки и состояние чтения, когда они доступны.
Настройки
Язык интерфейса, внешний вид, поведение ридера, обновление каталогов, управление данными, премиум-статус и юридические ссылки.

Reading surface

Статьи открываются во встроенном WebKit-ридере, а вокруг исходного документа добавляются элементы управления: тема, типографика, навигационные инструменты и офлайн-снимки для сохраненных материалов.

Интерфейс использует плоские поверхности, тонкие границы и сдержанный тон Foundation Terminal.

Content scope

Филиалы и каталоги

SCP Docs предоставляет точки входа для японского филиала, английской SCP Wiki, французского и русского филиалов, а также международного хаба там, где доступны данные каталогов. Манифесты каталогов загружаются как размещенные JSON-данные, кэшируются на устройстве и используются для списков и поиска.

  • Отчеты SCP — списки по сериям и разрешение названий с помощью метаданных.
  • Архивы разных форматов — Tales, GoI, Canon, Joke и связанные списки на основе манифестов.
  • Карты тегов — вспомогательные данные для поиска по тегам и классам объектов там, где они доступны.

Engagement

Состояние чтения и личная библиотека

С каждой URL статьи могут быть связаны прогресс прокрутки, личная оценка, избранное, статус «прочитать позже» и история на вашем устройстве. Это помогает возвращаться к чтению и строить личные библиотечные представления.

Премиум-возможности могут расширять архивный процесс офлайн-снимками, более широким поиском, заметками и повышенными лимитами сохранения. Актуальное предложение смотрите в описаниях внутри приложения и на экранах покупки.

Requirements

Системные требования

  • OS — сборки рассчитаны на iOS 17 и новее.
  • Сеть — требуется для обновления каталогов, онлайн-чтения статей, внешних страниц, рекламы и покупок.
  • Аккаунты — для использования приложения не требуется аккаунт SCP Foundation или Wikidot.

Legal

SCP Docs — неофициальное фанатское приложение. Исходные статьи, уведомления об авторских правах и условия лицензий принадлежат соответствующим авторам и исходным сайтам. Материалы, связанные с SCP, часто публикуются под Creative Commons BY-SA 3.0, но авторитетным источником остается исходная страница.
