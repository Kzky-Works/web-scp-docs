Feature Overview

Находите отчеты и действительно возвращайтесь к ним

SCP Docs объединяет точки входа, поиск и личное состояние чтения для материалов SCP в одном нативном iOS-ридере. Он создан для моментов после открытия: сохранить отчет, вернуться к недочитанной странице и перемещаться по большому архиву, не теряя контекст.

What it does

Основные возможности

Home

Точки входа по филиалам и категориям

Переходите к отчетам SCP, Tales, Canon, GoI, Joke и контекстам японского, английского, французского или русского филиала с одной стартовой поверхности.

Search

Многоуровневый поиск по архиву

Ищите по синхронизированным метаданным каталогов: названиям, тегам, классам объектов, состоянию чтения и заметкам там, где они доступны.

Library

Место, куда можно вернуться

Избранное, история, «прочитать позже» и личные оценки сохраняют ваш путь по архиву видимым на устройстве.

Reader

Сфокусированный просмотр статьи

Открывайте исходные страницы во встроенном ридере с типографическими настройками, обработкой темы и сохраненными снимками для подходящих материалов.

SCP Docs не является официальным приложением SCP Foundation или Wikidot. Исходные страницы, авторские указания и лицензионные уведомления на каждом исходном сайте остаются авторитетными.
